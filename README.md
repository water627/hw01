# hw01 Project

This project is focused on solving the N-Queens problem using Python. The project includes the implementation of the N-Queens solver, unit tests, and interactions with AI tools for debugging and refactoring.

## How to Run

1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/hw01.git
    ```
2. Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
3. Run the tests:
    ```bash
    python -m unittest discover
    ```

## AI Interaction Logs

Logs of interactions with AI tools will be recorded in the `prompt_log.md` file.
